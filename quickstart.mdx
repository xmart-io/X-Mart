---
title: 'Quickstart'
description: 'How to configure the Xmart account'
---

| Dato                                       | Información Necesaria                                                    | Descripción                                                                                                              | Utilidad                                                                                               |
|--------------------------------------------|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| **País** requerido                         | - Nombre                                                                 | Nombre del país donde operará la aplicación o la web.                                                                    | Necesario para la creación de direcciones de entrega, datos de facturación, etc.                       |
| **Permitir facturas como consumidor final** requerido | SI, NO                                                                | Mostrar formularios para el ingreso de datos de facturación                                                              | Solicitar datos de facturación antes de realizar la compra.                                            |
| **Ciudad(es)** requerido                   | - Nombre - Latitud - Longitud                                            | Listado de ciudades.                                                                               | En Pickup permite obtener las tiendas de la ciudad, permite ubicar la dirección de entrega.            |
| **Sector(es)** requerido                   | - Nombre - Latitud - Longitud - Ciudad                                   | Listado de sectores.                                                                                | Junto con la ciudad permite ubicar la dirección de entrega, necesario para crear direcciones de entrega.|
| **Marca(s)** requerido                     | - Nombre de la marca (Vendor) - Descripción breve de la marca - Identificador único de la marca | Listado de marcas.                                                                                 | Necesario para la creación de tiendas, canales de venta y listas.                                      |
| **Logo(s) de marca(s)** opcional           | Archivo comprimido con los logos de las marcas (jpg/png), cada archivo de imagen debe tener el nombre de la marca. | Archivo comprimido de imágenes de marcas.                                                                                | En futuras versiones puede mostrarse el logo de la marca.                                              |
| **Zonas** requerido                        | Archivos kml, kmz, csv                          | Archivos que reflejen los polígonos de las zonas de cobertura.                                                           | Necesario para obtener la tienda más cercana.                                                          |
| **Listado de zonas y tiendas** requerido   | - Nombre, identificador o código de la tienda - Zona de cobertura - Ciudad | Listado que refleje la relación entre zona de cobertura y tienda.                                   | Listado utilizado por Trade para la configuración de zonas por tienda.                                 |


"NOTE: ONCE THE ACCOUNT IS CONFIGURED, THE URL OF THE ENVIRONMENT TO POINT TO AND THE CREDENTIALS OF THE NEW OAUTH CLIENT WILL BE NOTIFIED VIA EMAIL."
